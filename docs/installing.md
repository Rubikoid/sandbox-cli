# Install sandbox-cli

`sandbox-cli` is availabe as [sandbox-cli](https://pypi.org/project/sandbox-cli/) on PyPi.

Can be installed with `pipx` (recommeded) or `pip`:

```shell
# with pipx
pipx install sandbox-cli

# with pip
pip install sandbox-cli
```
